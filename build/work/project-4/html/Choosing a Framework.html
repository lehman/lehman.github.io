Choosing a Framework

Frameworks are built on top of programming languages. There are dozens, or even hundreds, of frameworks in multiple programming languages. The best way to pick a web framework that works best for you is to see if your favorite scripting languages has a framework available. If you don't have a favorite scripting language, or don't know what that is, go ahead and just browse these popular frameworks:

  • Ruby: Ruby on Rails, Merb
  • Python: Django, Pylons, Turbo Gears
  • PHP: CakePHP, Symfony, Zend, CodeIgniter, Drupal
  • Perl: Catalyst
  • Java: Apache Click, AppFuse, Google Web Toolkit, Play!
  • ColdFusion: Fusebox, Model Glue, Mach-II
